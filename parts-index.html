아누비스의 심판 — 부품 라이브러리

아트 스타일 가이드를 따라 4개의 핵심 SVG 부품을 분리 제작했습니다. 각 부품은 <g id="..."> 으로 그룹화되어 코드에서 개별 변형/회전/색상 변경이 가능합니다. 아래 인터랙티브 데모로 회전·분수·숫자 주입을 직접 시험해 보세요.

02 · HEART · ITEM

심장 (분수 그리드)
parts/heart-item.svg · 16칸 (4×4) 그리드

부품 ID

  • #heart-shell — 심장 외곽
  • #heart-grid — 분수 그리드
  • #heart-cell-rXcY — 개별 셀 (r,c = 0..3)
  • #heart-glow — 빛 (선인 연출)

분수 시각화

사용 예시 (JS) 
// 분수 N/16 시각화
function setFraction(n) {
  for (let i = 0; i < 16; i++) {
    const r = Math.floor(i/4), c = i%4;
    const cell = svg.querySelector(`#heart-cell-r${r}c${c}`);
    cell.classList.toggle('filled', i < n);
    cell.classList.toggle('empty', i >= n);
  }
}
setFraction(12); // = 12/16 = 3/4

// (visual: positive angle = 시계방향, 우측이 내려감)
